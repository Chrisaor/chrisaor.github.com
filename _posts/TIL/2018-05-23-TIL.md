#### Sequential Sort

```python
def sequential_search(string, inputstr):
    index = 0
    while 1:
        if index == len(string):
            return -1
        elif inputstr in string[index]:
            return index
        else:
            index += 1

string = 'abcdefg'
inputstr = 'z'
print(sequential_search(string,inputstr))
```
