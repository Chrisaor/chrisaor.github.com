#### 새 키페어 생성 후 pem파일 다운로드

다운로드 한 파일을 ~/.ssh 폴더로 이동시킨다.

#### pem 파일 권한 재설정

```
chmod 400 ~/.ssh/<name>.pem
```

#### 우분투 서버 접속(?)

```
ssh -i ~/.ssh/<name>.pem ubuntu@<public DNS>
```

#### pipenv설치

#### pyenv설치

```

#### vi ~/.zshrc에서 맨 아래에 추가

```

eval "$(pyenv init -)"
eval "$(pyenv virtualenv-init -)"
```

```
