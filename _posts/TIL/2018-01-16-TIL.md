# Git

## 브랜치

Git의 브랜치는 커밋 사이를 가볍게 이동할 수 있는 어떤 포인터 같은 것이다.

```
$ git branch testing
```

새 브랜치 생성하기

```
$ git checkout testing
```
 브랜치 이동하기

브랜치를 이동하면 HEAD는 testing 브랜치를 가리키게 되는데 이 때 HEAD는 현재 작업하는 로컬 브랜치를 가리키는 특수한 포인터다.
