**책 내용에 대한 무단배포 방지를 위해 최소한으로 정리**

```python
txt1 = 'A'
txt2 = '안녕'
txt3 = 'Warcraft Three'
txt4 = '3PO'
print(txt1.isalpha())
print(txt2.isalpha())
print(txt3.isalpha())
print(txt4.isalpha())
```
