**책 내용에 대한 무단배포 방지를 위해 최소한으로 정리**


```python
x = 1;y = 2
str1 = 'abc';str2 = 'python'
print(x==2)
print(x!=y)
print(str1==str2)
print(str2=='python')
print(str1<str2)
```

str1과 str2는 사전 순서를 기준으로 비교한다.
