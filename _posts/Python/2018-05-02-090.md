**책 내용에 대한 무단배포 방지를 위해 최소한으로 정리**


```python
num1 = 1234
num2 = 3.14

numstr1 = str(num1)
numstr2 = str(num2)
print(f'num1을 문자열로 변환한 값은 {numstr1})
print(f'num2를 문자열로 변환한 값은 {numstr2})
```
