**책 내용에 대한 무단배포 방지를 위해 최소한으로 정리**

```python
numdata = 57
strdata = '파이썬'
listdata = [1,2,3]
dictdata = {'a':1,'b':2}

def func():
    print('hi')

print(type(numdata))
print(type(strdata))
print(type(listdata))
print(type(dictdata))
print(type(func))
```
