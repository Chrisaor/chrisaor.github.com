**책 내용에 대한 무단배포 방지를 위해 최소한으로 정리**

```python
txt = 'A lot of things occur each day, every day.'
word_count1 = txt.count('o')
word_count2 = txt.count('day')
word_count3 = txt.count(' ')
print(word_count1)
print(word_count2)
print(word_count3)
```
