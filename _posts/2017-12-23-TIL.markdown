7번째 문제인 WaterMelon을 성공해서 제출하고 다른 사람의 풀이과정을 보는데 입이 떡 벌어짐...

나의 코드 :
```python
def water_melon(n):
	str = ''
	for i in range(n):
		if i % 2 == 0:
			str += '수'
		else:
			str += '박'
	return str


# 실행을 위한 테스트코드입니다.
print("n이 3인 경우: " + water_melon(3));
print("n이 4인 경우: " + water_melon(4));
```

추천을 가장 많이 받은 코드 :

def water_melon(n):
    s = "수박" * n
    return s[:n]
