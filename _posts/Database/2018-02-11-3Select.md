## The SQL SELECT Statement

SQL에서 가장 많이 사용되는 명령어

데이터를 선택하고 가져오는 역할

**SQL SELECT Syntax**

특정 필드의 값 가져오기

```sql
SELECT CustomerName, City
From Customer;
```
