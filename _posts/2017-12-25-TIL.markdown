알고리즘 연습 : 문자열에 p와 y의 개수를 비교해 같은 True, 다르면 False 리턴하기
lower() = 모두 소문자로 만들기 , upper() = 모두 대문자로 만들기
count() = 인수와 같은 값의 개수 세기

나의 코드 

```python
def numPY(s):
    pNum = 0
    yNum = 0
    for i in range(len(s)):
        if s[i] == "p" or s[i] =="P":
            pNum += 1
        elif s[i] == "y" or s[i] =="Y":
            yNum += 1
    if pNum == yNum:
        return True
    else:
        return False

print(numPY("pPoooyY"))
print( numPY("Pyy") )
```

고수의 코드

```python
def numPY(s):
    # 함수를 완성하세요
    return s.lower().count('p') == s.lower().count('y')

# 아래는 테스트로 출력해 보기 위한 코드입니다.
print( numPY("pPoooyY") )
print( numPY("Pyy") )
```
