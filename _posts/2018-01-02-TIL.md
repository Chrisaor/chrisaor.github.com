### Zsh
- 기본적으로는 bash이지만 Zsh이 제일 편리함
```
$ brew install zsh zsh-completions
$ curl -L http://install.ohmyz.sh | sh
$ chsh -s `which zsh`
//chsh : /usr/local/bin/zsh : non-standard shell 오류 발생시
//$ sudo vim /etc/shells 아래 쪽에 /usr/local/bin/zsh 추가하고 :wq 저장
```
